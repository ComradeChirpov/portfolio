Gulag Life

(CSE, 8/23/2017)

Gulag Life is a Scratch game about a cat who is attempting to navigate his way through a new day job in a gulag. The project was assigned in CSE with the objective of getting students to understand and learn how to code, and challenges included an unproductive partner, frequent debugging, and mandatory documentation. The game itself is noted for its semi-communist humor and light-hearted tone but harsh consequences when the player fails a task... or disobeys orders (Don't worry, no cats were harmed in the production of the game).


HTML Project

(CSE, 11/28/2017)

This website is a HTML project to serve as an example to show my grasp of HTML and CSS, as well as the Bootstrap responsive design framework. The project was assigned in CSE to help students learn the basics of web development and responsive design. Several challenges I had to overcome were mostly learning the new language, debugging the code routinely, and organizing code into a readable fashion. I consider this website to be a competent one that is reasonably well-developed but I realize there is still room for improvement. (Clicking the picture will bring you back to the home page).


GPS Accuracy Logger

(CSE, 10/6/2017)

The App Inventor project was an interesting one that introduced me to creating apps on the MIT App Inventor system. It was the first time that I had to integrate a phone or tablet's GPS/Location sensors into a program that could display the user's latitude and longitude and even indicate how accurate the device was at displaying location.
